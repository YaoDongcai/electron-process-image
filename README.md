> This is a Electron-vue 项目
# 建议用yarn来安装 npm安装后会有问题
# yarn install 安装后 =>
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


# lint all JS/Vue component files in `src/`
npm run lint

```
